# NICE

## Sobre o Projeto

O **NICE** é um aplicativo voltado para o apoio à saúde emocional e ao bem-estar mental dos usuários.

A proposta do projeto é oferecer uma plataforma simples e acessível para auxiliar pessoas no acompanhamento de sua jornada emocional, promovendo autoconhecimento, organização e continuidade no cuidado com a saúde mental.

## Objetivo

O objetivo do NICE é ajudar usuários a acompanharem seu estado emocional por meio de recursos digitais, oferecendo uma experiência prática e intuitiva para o registro, consulta e visualização de informações relacionadas ao bem-estar.

## Público-Alvo

O sistema foi pensado para atender diferentes tipos de usuários:

- Pacientes;
- Psicólogos;
- Empresas.

Cada perfil possui uma experiência adequada às suas necessidades dentro da plataforma.

## Tecnologias e Estrutura

O projeto é composto por uma aplicação mobile integrada a serviços de backend, banco de dados, autenticação e recursos de análise de informações.

A arquitetura foi planejada para garantir organização, segurança e boa experiência de uso.

## Finalidade Acadêmica

Este projeto foi desenvolvido como parte das atividades acadêmicas da disciplina de Engenharia de Software, com foco em documentação, arquitetura, qualidade e testes de software.

## Conclusão

O NICE busca unir tecnologia e saúde mental em uma solução simples, moderna e organizada, contribuindo para o acompanhamento emocional dos usuários e para a melhoria contínua do cuidado com o bem-estar.
